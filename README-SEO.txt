- cartes Twitter

- données structurées Organization
- données structurées FAQPage
- données structurées dynamiques pour les thèmes et les fiches
- titres et descriptions dynamiques sur les pages de thèmes et de douaas

Installation :
1. Remplacer tout le projet par le contenu de cette archive.
2. Faire un commit et un push sur GitHub.
3. Vérifier :
   https://douaagenerator.fr/robots.txt
   https://douaagenerator.fr/sitemap.xml
4. Dans Google Search Console :
   Indexation > Sitemaps
   Ajouter : sitemap.xml

Important :
- Les pages dynamiques utilisent encore des URLs avec ?id=.
- Une étape SEO ultérieure pourra créer des URLs statiques lisibles.
- Version de cache : v18
